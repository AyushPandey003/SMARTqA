# UI/UX Guidelines

## Color Palette
- **Primary Button (Add to Cart)**: Blue (#007bff).
- **Action Button (Pay Now)**: Green.
- **Error Messages**: Red text.
- **Success Messages**: Green text, bold.

## Form Interaction
- Validation errors should appear inline below the invalid field.
- The "Pay Now" button should be disabled after a successful payment to prevent double submission.
- Discount code feedback should be immediate (no page reload).
