UI/UX Guidelines

1. Error Messages
   - All form validation errors must be displayed in red text.
   - Error messages should be inline, below the invalid field.

2. Buttons
   - The "Pay Now" button should be green (#008000).
   - Secondary buttons like "Update" or "Apply" should be default gray or styled neutrally.

3. Success Feedback
   - Upon successful payment, replace the form with a message "Payment Successful!" in green text.
   - Discount code success message should be "Discount Applied!" in green.
